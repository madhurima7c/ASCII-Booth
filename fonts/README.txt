PP BitNeu (title font)
----------------------
This folder includes webfont files used as "PP BitNeu" in styles.css:

  PPBitNeu.woff2
  PPBitNeu.woff
  PPBitNeu.ttf

They were copied from Pangram Pangram’s **PP Neue Bit Bold** family
(PPNeueBit-Bold.* in the “Bitmap Fonts” folder of the starter pack).
Use only under your Pangram license.

To switch weight, replace these files with PPNeueBit-Regular.* from the
same folder (still rename to PPBitNeu.*), or add a second @font-face in
styles.css.
